# Story Outline

Three-act structure. The eras are the second act — the bulk of the game. Each era is a chapter.

---

## Act One — Home

**Goal:** Establish what Caret has, so losing it means something.

### Scene 01: A Beautiful Homepage
*[[scenes/01-a-beautiful-homepage]]*
- Caret at home. Tutorial mechanics through free exploration.
- Introduces `>`, the relationship, the vibe.
- Establishes the page as alive, semantic, intentional.
- Plants the glitching element. Caret touches it anyway.

### Scene 02: The Rift
*[[scenes/02-the-rift]]*
- The page de-renders in reverse order: dark mode, custom properties, grid, floats, border-radius, fonts.
- `>` is separated. The last thing Caret hears is their voice through static.
- White. Times New Roman. 1991.

**End of Act One**

---

## Act Two — The Journey

*Nine chapters, one per era. Each follows roughly the same structure: arrival, exploration, boss, victory reflow.*

*Interspersed: friend reunions. Each reunion can happen in any order within the era they're set in. The party grows.*

### Chapter 1: The Document (Era 1, 1991–1996)
*[[scenes/03-era-1-arrival]]*
- Arrive in Times New Roman. First contact with the Webmaster.
- Mechanics: heading staircase, `<blink>` timing, `<table>` layout puzzles, `<hr>` platforms.
- Friend reunion: `/` (Slash) — found trying to close an unclosed `<p>` tag that has been open since 1993.
- Boss: `<font>` — The Inline Styler. Not malicious. Just a tool given too much responsibility.
- Victory: CSS unlocked. Color arrives. The world gets intentional for the first time.

### Chapter 2: Geocities (Era 2, 1996–2001)
- Tiled backgrounds, MIDI chaos, sparkle cursor trails. Beautiful disaster.
- Mechanics: animated GIF NPCs, `<center>` gravity wells, web rings as loops.
- Friend reunion: `>` (Chevron) — stuck in a web ring for four eras.
  *[[scenes/04-reunion-chevron]]*
- Friend reunion (preview): Nyan Cat spotted riding a `<marquee>`, not yet freed.
- Boss: Presentation Hydra (`<center><font><blink>`) — three deprecated tags fused. Humor + pathos.
- Victory: Semantic structure arrives. HTML gains meaning separate from appearance.

### Chapter 3: CSS Zen Garden (Era 3, 2001–2006)
- Same HTML, completely different look. The revelation of separation of concerns.
- Mechanics: floats, `z-index` stacking, `overflow: hidden`, position: absolute chaos.
- Friend reunions: `#` Hash (broken anchor chain in IE6 chaos), `;` Semicolon (forgotten at end of a `color:` declaration).
- Boss: IE6 — The Incompatible. Not fought, navigated. A gauntlet of rendering bugs.
- Post-boss: IE6 doesn't die, just diminishes. First appearance of the IE6 ghost.
- Victory: JavaScript arrives. Elements wake up.

### Chapter 4: The jQuery Era (Era 4, 2006–2012)
- Rounded corners. Web 2.0. `.animate()` everything. The golden age of plugins.
- Mechanics: `.slideDown()` platforms, `.fadeIn()` / `.fadeOut()`, `.draggable()` (Pointer mode unlocks here).
- Friend reunion: `*` Asterisk (reset their own identity with a CSS reset, wandering).
- Friend reunion: `&` Ampersand (stuck in an encoding loop).
- Companion unlock: **Clippy** — trapped in a `<div class="ui-helper-hidden">`.
  *[[scenes/05-clippy]]*
- Boss: `$.fn.modal()` — The Plugin Monolith. Modals within modals within modals.
- Victory: Native HTML elements gain powers. `<dialog>`, `<details>`, `<progress>` arrive.

### Chapter 5: Responsive Revolution (Era 5, 2012–2015)
- Bootstrap. Mobile-first. The viewport changes size mid-level.
- Mechanics: breakpoint shifts, hamburger nav reveal, flexbox alignment.
- Friend reunion: `☰` Hamburger — collapsed by a media query, never reopened.
- Companion unlock: **BonziBuddy** — found in a legacy guestbook (still operational, somehow).
- Boss: Bootstrap Grid Hydra — layout instability incarnate.
- Victory: Accessibility becomes visible. ARIA landmarks. The a11y tree unlocks as a mechanic.

### Chapter 6: Component Era (Era 6, 2015–2019)
- Material Design. Elevation. Ripples. Card-based everything.
- Mechanics: elevation z-axis, shadow-based depth, CSS Grid arrives, Web Components.
- Companion unlock: **Dancing Baby** — freed from an `<embed>` tag with `autoplay` and `loop` on.
- Boss: `node_modules/` — The Dependency Black Hole. Performance-as-combat.
- BonziBuddy subplot: the boss tries to recruit him back into the adware network. Loyalty arc.
- Victory: CSS Grid. `fr` units. True 2D layout.

### Chapter 7: The Tailwind Age (Era 7, 2019–2024)
- shadcn. Utility-first. The current era — familiar ground, slightly uncanny.
- Mechanics: dark mode toggle changes physics, cn() ability merging.
- Companion unlock: **Nyan Cat** — `<marquee>` finally deprecated, free to move both directions.
- Boss: Utility Sprawl — a class string that became sentient.
- Victory: Container queries. `@layer`. `@scope`. CSS is self-sufficient.

### Chapter 8: The Spatial Web (Era 8, 2024–????)
- 2.5D. Scroll-driven animations. Glassmorphism. The web finally escapes flat rectangles.
- Mechanics: CSS perspective, `:has()` selector reactions, anchor positioning, Popover API.
- The Dancing Baby reaches Pixar-quality rendering. The contrast is enormous.
- Boss: The Obfuscator — `<div>` soup made of hashed class names. Semantic replacement puzzle.
- Victory: `view-source:` restored. The final boss preview: a corrupted homepage ahead.

### Chapter 9: The Source (Endgame)
- `view-source:`. Green text on black. The level IS the source code.
- Mechanics: platform on `<` and `>` characters, fix markup to change the compiled level.
- The final companion arc: the Dancing Baby finally gets to stop dancing.
- Boss: The Malformed Document — not combat, repair.
- The Clippy sacrifice: "It looks like you're trying to fix everything. Let me help." He becomes the `/>` that closes the void element.
- Victory: The homepage reloads. Caret is home. Everything looks the same. Nothing is the same.

**End of Act Two**

---

## Act Three — Return

### Scene: Homecoming
- The modern homepage. Era 7's visual language. Everything is where it was.
- Caret is back. The party is here — `>`, all the symbol friends, the companions.
- The page looks the same. The DOM Inspector shows everything differently now.
- Caret reads the `<nav>` label. Reads the `<aside>` label. Reads the `<button>` label.
- They understand all of it. Not intellectually. In their bones.

### Final Scene: The Credits
- Credits play as a perfectly structured semantic HTML document.
- Every tag used correctly. Headings that actually create hierarchy. Links with meaningful text. Images with alt attributes. A `<main>`, a `<nav>`, an `<aside>` with complementary content.
- Caret navigates the credits page like a level.
- At the very end: a single blinking `│` on an empty line.
- The cursor blinks. It's home. It understands what it's blinking in.
- Fade.

## Emotional Arc

| Point | Caret's State |
|-------|--------------|
| Opening | Comfortable, unthinking. Takes the web for granted. |
| Rift | Disoriented. Stripped of everything familiar. |
| Era 1 | Struggling. Curious despite themselves. |
| Era 2-3 | Adapting. Starting to find the beauty in constraints. |
| Era 4-5 | Growing. Collecting friends, building understanding. |
| Era 6 | Tested. BonziBuddy's loyalty arc forces a question: does history define you? |
| Era 7 | Almost home. The familiar feels uncanny. |
| Era 8-9 | Clear-eyed. Understanding everything. The source is visible. |
| Return | Home. The same page. Different Caret. |
